## Installation

[Node.js](https://nodejs.org/) and NPM are required to run.

```bash
# install dependencies
> `npm i`  ou `yarn add`
```

## Production
```bash
# Create Build and start the application
> `npm run dev`  ou `yarn  dev`
```
## Development
```bash
# , Create Build 
> `npm run build`  ou `yarn build`

# Create Build and start the application
> `npm run start`  ou `yarn start`
```
